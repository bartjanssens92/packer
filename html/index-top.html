Boxes

Boxes.bbqnetwork.be is a collection of vagrant boxes. Here they are served for vagrant to use.

They are build weekly using packer from this repository.

Add the following lines in the Vagrantfile to start using the boxes: 

      config.vm.box = "arch-amd64-virtualbox"
      config.vm.box_url = "https://boxes.bbqnetwork.be/arch-amd64-virtualbox"
